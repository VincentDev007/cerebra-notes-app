# Cerebra

![version](https://img.shields.io/badge/version-0.5.0-blue)
![platform](https://img.shields.io/badge/platform-macOS-lightgrey?logo=apple)

Desktop notes app with full-text search and sticky notes.

> macOS only (12 Monterey or later). Windows and Linux support is not planned for this release.

## Features
- Notes and folders
- Sticky notes
- Full-text search
- Dark mode
- Persistent tabs
- Settings

## Download
Get the latest release [here](https://github.com/VincentDev007/cerebra-notes-app/releases/tag/v0.5.0).

## Running from Source
1. Clone the repo
2. Navigate to the project folder
   ```
   cd cerebra-software
   ```
3. Install dependencies
   ```
   npm install
   ```
4. Start the app
   ```
   npm run dev
   ```

## Tech Stack
- Electron
- React
- TypeScript
- SQLite (better-sqlite3)
