# Cerebra

A local-first note taking app built with Electron.

## Features
- Notes and folders
- Sticky notes
- Dark mode
- Persistent tabs
- Settings

## Getting Started

1. Clone the repo
2. Navigate to the project folder
   ```
   cd cerebra-software
   ```
3. Install dependencies
   ```
   npm install
   ```
4. Start the app
   ```
   npm run dev
   ```

## Tech Stack
- Electron
- React
- TypeScript
- SQLite (better-sqlite3)
